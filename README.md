This repository contains an R script created for a live coding session as part of the [Data Scientist (Curves)](https://www.linkedin.com/jobs/view/4252932917/) application process at [Argus Media Group](https://www.argusmedia.com), a leading independent provider of global energy and commodity market intelligence.
